How to Join Skythoria

Address: Skythoria.ooguy.com

Step 1: Open the Minecraft launcher, go to the Installations tab, and create a new installation. Choose 1.21.6 from the version dropdown, then click Install.

Step 2: Be sure to select release 1.21.6 from the version list before launching.

Version 1.21.6 should now appear correctly installed with the furnace icon.

Step 3: In Minecraft, go to Multiplayer → Add Server. Type Skythoria.ooguy.com into the address box, name it anything, and click Done.

Requirements

  • Minecraft Java Edition 1.21.6
  • English Language
  • Java (not Bedrock)
